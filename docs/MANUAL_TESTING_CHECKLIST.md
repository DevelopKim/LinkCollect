### 3. 상대 경로/절대 경로 혼합 페이지 테스트

#### 테스트용 HTML 파일 생성 필요
```html
<!DOCTYPE html>
<html>
<head><title>테스트 페이지</title></head>
<body>
  <a href="/relative1">상대 경로 1</a>
  <a href="../parent">상대 경로 2</a>
  <a href="https://example.com/absolute">절대 URL</a>
  <a href="//example.com/protocol-relative">프로토콜 상대</a>
</body>
</html>
```

- [ ] 모든 링크가 정상적으로 절대 URL로 변환됨
- [ ] 상대 경로가 현재 페이지 기준으로 올바르게 변환됨
